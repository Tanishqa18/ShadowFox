TaskMate

TaskMate is a user-friendly task management website designed to help users organize, track, and manage their daily tasks efficiently. The platform allows users to create, update, prioritize, and categorize tasks, set deadlines, and monitor progress through a centralized dashboard. ChatBot

This project is an AI-powered chatbot website that generates human-like responses using a pre-trained language model. The chatbot is built using Python and Flask for the backend, with Hugging Face Transformers and PyTorch for natural language processing. It provides an interactive chat interface where users can input text and receive dynamic responses in real time.
